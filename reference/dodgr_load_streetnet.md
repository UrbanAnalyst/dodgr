# Load a street network previously saved with [dodgr_save_streetnet](https://UrbanAnalyst.github.io/dodgr/reference/dodgr_save_streetnet.md).

This always returns the full, non-contracted graph. The contracted graph
can be generated by passing the result to
[dodgr_contract_graph](https://UrbanAnalyst.github.io/dodgr/reference/dodgr_contract_graph.md).

## Usage

``` r
dodgr_load_streetnet(filename)
```

## Arguments

- filename:

  Name (with optional full path) of file to be loaded.

## Value

The loaded street network.

## See also

Other cache:
[`clear_dodgr_cache()`](https://UrbanAnalyst.github.io/dodgr/reference/clear_dodgr_cache.md),
[`dodgr_cache_off()`](https://UrbanAnalyst.github.io/dodgr/reference/dodgr_cache_off.md),
[`dodgr_cache_on()`](https://UrbanAnalyst.github.io/dodgr/reference/dodgr_cache_on.md),
[`dodgr_save_streetnet()`](https://UrbanAnalyst.github.io/dodgr/reference/dodgr_save_streetnet.md)

## Examples

``` r
net <- weight_streetnet (hampi)
f <- file.path (tempdir (), "streetnet.Rds")
dodgr_save_streetnet (net, f)
clear_dodgr_cache () # rm cached objects from tempdir
# at some later time, or in a new R session:
net <- dodgr_load_streetnet (f)
```
